# Salut, moi c’est Dimitri 👋

Je suis étudiant en BTS SIO option SLAM, et je m’intéresse surtout au développement web et applicatif.

J’aime particulièrement le back-end, surtout avec C# / .NET, parce que j’aime comprendre la logique derrière une application, construire des API, manipuler les données, et faire en sorte que tout fonctionne proprement.  
En parallèle, je continue aussi de progresser sur le front-end avec React et JavaScript pour devenir de plus en plus autonome sur des projets complets.

Ce que j’aime dans le développement, c’est surtout apprendre en faisant.  
Tester, me tromper, corriger, recommencer, comprendre pourquoi ça ne marche pas, puis finir par trouver une solution. C’est vraiment comme ça que j’avance et que je progresse.

Au fil de mes projets, j’ai eu l’occasion de travailler sur plusieurs aspects du développement :
- création d’API en C# / .NET
- développement d’interfaces web
- gestion de bases de données SQL Server
- utilisation de Git, GitHub et Azure DevOps
- documentation technique, organisation de projet et débogage

J’essaie de construire des projets concrets, utiles, et surtout de mieux comprendre les bonnes pratiques à chaque étape.  
Mon objectif, ce n’est pas juste de faire du code qui marche, mais de faire du code que je comprends, que je peux améliorer, et dont je peux être fier.

## Technologies que j’utilise
- C# / .NET
- React
- JavaScript
- HTML / CSS
- SQL Server
- Git / GitHub / Azure DevOps

Merci d’être passé sur mon profil.
